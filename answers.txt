
Q1: Why might be consider using git?

Answer: The purpose of Git is to manage source code. A developer might consider using git in order to clearly organize and keep track of the changes made to their project, so if something ever breaks they can go back to prior versions. 


Q2: What field could data science be applied to? Give and Example and how it might help

Answer: Data Science is a subject which can be applied to many different fields. One such example is the field of healthcare, in which you can use data science to analyze various datasets from clinical trials or patient data to see what types of patients may be more susceptible to certain diseases.
